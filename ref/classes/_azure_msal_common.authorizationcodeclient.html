getAuthCodeUrl

  • getAuthCodeUrl(request: CommonAuthorizationUrlRequest): Promise<string>
    • Defined in msal-common/src/client/AuthorizationCodeClient.ts:52

    Creates the URL of the authorization request letting the user input credentials and consent to the application. The URL target the /authorize endpoint of the authority configured in the application object.

    Once the user inputs their credentials and consents, the authority will send a response to the redirect URI sent in the request and should contain an authorization code, which can then be used to acquire tokens via acquireToken(AuthorizationCodeRequest)
